# Pokemon Bot

A Slack bot made with a passion for Pokémon. It was created for stardance.space and is installed on Hack Club.

## Features

- `/pb-help` - Show available commands
- `/pb-pokemonfact [pokemon name or id]` - Get a fun fact about a Pokémon
- `/pb-berrydeets [berry name]` - Get a fun fact about a Berry
- `/pb-list-berries` - List berries you can ask about
- `/pb-ping` - Check the bot's latency
- `/pb-joke` - Get a random joke

## How to use

Head over to The Hack Club Slack and run `/pb-help` !

## Notes

- The bot uses [Slack Bolt](https://slack.dev/bolt-js/) and [PokeAPI](https://pokeapi.co/).
- Pokémon and berry data come from public APIs, so command output depends on network access.
